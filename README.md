# 🚢 Titanic Dataset — Exploratory Data Analysis

A beginner-friendly EDA project on the famous Titanic dataset to uncover patterns and insights about passenger survival.

---

## 📌 About the Project

The Titanic dataset is one of the most well-known datasets in Data Science.  
In this project, I explore the data using Python to answer questions like:

- What was the overall survival rate?
- Did gender or age affect survival chances?
- Which passenger class had the highest survival rate?
- How did fare prices vary across classes?

## 📊 Key Findings

- 🔹 Overall survival rate was approximately **38%**
- 🔹 **Women** had a significantly higher survival rate than men
- 🔹 **1st class** passengers survived at a much higher rate than 2nd and 3rd class
- 🔹 **Children** (age < 10) had better survival chances
